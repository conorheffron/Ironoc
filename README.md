# About
Personal website / portfolio  http://www.ironoc.com/

# Tech Stack
Java 8, Spring Boot 1.4.3, HTML5+CSS, JQuery

# Run
Build Image:
```
mvn clean package docker:build 
```

Spin-up Container: 
```
docker-compose up -d
```

Tear-down:
```
docker-compose down
```
